## Overview

This 8-week intensive bootcamp is designed to transform beginners into job-ready cloud platform engineers. Each week builds upon the previous, with daily lessons, hands-on labs, and weekly assessments.

**Total Duration**: 8 weeks  
**Daily Commitment**: 6 hours (structured sessions)  
**Weekly Assessment**: 2-3 hours  
**Total Hours**: ~250 hours  

---

## Week 1: Linux Fundamentals & Command Line Mastery

### Goal
Master the Linux command line, file system operations, and basic system administration tasks.

### Learning Objectives
- Navigate the Linux file system with confidence
- Manage files, directories, and permissions
- Understand processes and system resources
- Use essential command-line tools
- Perform basic system administration

### Daily Breakdown

#### Day 1: Linux Introduction & File System
- **Morning**: Linux history, distributions, file system hierarchy
- **Afternoon**: Navigation (cd, ls, pwd), file operations (cp, mv, rm, touch)
- **Evening**: Practice exercises and quiz
- **Skills**: File navigation, basic file operations

#### Day 2: File Permissions & Ownership
- **Morning**: Understanding permissions (rwx), users and groups
- **Afternoon**: chmod, chown, chgrp, umask in practice
- **Evening**: Permission troubleshooting scenarios
- **Skills**: Permission management, security basics

#### Day 3: Text Processing & Filters
- **Morning**: grep, sed, awk, cut, sort, uniq
- **Afternoon**: Building text processing pipelines
- **Evening**: Log analysis challenges
- **Skills**: Text manipulation, log analysis

#### Day 4: Process Management
- **Morning**: Process lifecycle, ps, top, htop
- **Afternoon**: kill, nice, background jobs, systemd
- **Evening**: Process troubleshooting
- **Skills**: Process control, system monitoring

#### Day 5: Package Management & System Updates
- **Morning**: apt/yum/dnf, package repositories
- **Afternoon**: Installing, updating, removing software
- **Evening**: Dependency resolution challenges
- **Skills**: Package management, system maintenance

### Week 1 Assessment
**Duration**: 2 hours  
**Format**: Practical tasks + scenarios  
**Topics**: File operations, permissions, text processing, processes  
**Passing Score**: 70%

## Tips for Success

1. **Stay Consistent**: Follow the daily schedule
2. **Hands-On Practice**: Theory is important, but practice is essential
3. **Ask Questions**: Use the mentor agent liberally
4. **Build Projects**: Apply what you learn immediately
5. **Document Everything**: Keep notes and create runbooks
6. **Join Communities**: Engage with DevOps/Platform engineering communities
7. **Stay Curious**: Explore beyond the curriculum
8. **Practice Troubleshooting**: Break things and fix them
9. **Review Regularly**: Revisit previous weeks' content
10. **Focus on Job Skills**: Think about interview questions throughout
